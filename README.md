# eye_diseases
 CNN classifier project

Classifies images of eyes into:
1. cataracts
2. jaundice
3. uveitis
4. normal

Original dataset: https://www.kaggle.com/datasets/kondwani/eye-disease-dataset (was heavily modified!)
